## Introduction

This is a IoT simple project, there a 2 controller communicate using lorawan (433mhz frequency).

Controller 1 is a Transmitter and Controller 2 is a Receiver

Inside Controller 1, im adding a Web Server and Action to send a packet to Controller 2. That a string packet to turn on / off a light connected with Controller 2.
